Sukhumwit Crawler Menus

File

  • New Contact...: Allows the manual addition of a contact to the list of Contacts Found.
  • Save Contacts As...: Prompts for a file to save the Contacts List to, then saves the list as a CSV file.

Theme

  • Selecting from the Theme menu allows you to change the color theme of the application. The default theme (Steel) and several other themes are included. One note that the themes can control the sizes, as well as the colors of many controls. Also included is the ability to read a theme's colors from a text file.

Help

  • About: This menu item brings up a dialog with a brief description of the application.
  • Help: brings up this screen which displays a set of HTML files containing all sorts of useful info.
